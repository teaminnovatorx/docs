# ADR-001: Tech Stack Selection

**Status:** Proposed
**Date:** 2026-05-23
**Deciders:** TeamInnovatorX

---

## Context

We need to select a tech stack for our hackathon project. The team has diverse skill sets and the project needs to be built in 6 weeks.

---

## Decision

To be decided at the May 23 meeting.

---

## Options to Evaluate

| Stack | Frontend | Backend | Database | Hosting |
|-------|----------|---------|----------|---------|
| Option A | Next.js | Node.js/Express | PostgreSQL | Vercel + Railway |
| Option B | React | FastAPI | MongoDB | Fly.io |
| Option C | Next.js | Next.js API Routes | Supabase | Vercel |
| Option D | Vue.js | Django | PostgreSQL | Render |

---

## Evaluation Criteria

- Team familiarity
- Learning curve
- Scalability
- Free tier availability
- Community support
- Development speed
