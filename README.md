# Team InnovatorX Docs

Documentation hub for Team InnovatorX — synced with HackMD.

## Structure

```
docs/
├── meetings/          # Meeting notes (auto-synced from HackMD)
├── architecture/      # System design, ADRs
├── research/          # Problem discovery, tech evaluation
├── retrospectives/    # Sprint retrospectives
├── decisions/         # Architecture Decision Records
└── templates/         # Reusable templates
```

## How It Works

1. Write notes in HackMD (shared link, real-time collab)
2. Push to this repo via HackMD GitHub integration
3. All docs version-controlled in one place
